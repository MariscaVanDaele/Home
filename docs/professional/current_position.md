# Learning Developer at TU Delft

In December 2025 I started as a Learning Developer in the Learning for Life Centre of TU Delft. 
The Learning for Life Centre is responsible for the Massive Open Online Courses (MOOCs) and Professional Education courses the various faculties create for professionals. 
As a Learning Developer, I support the course teams in creating and maintaining these courses. I also investigate new options, about which I might tell more in the future. 

Remember, I'm still setting up this website, so part of what I'm doing is 'just making documents' to learn how to do that.
